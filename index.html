About this website.

Hello, my name is Alexander. I like album covers.

I believe cover artwork is a very essential part of releasing a work of music, because it may influence how the listener perceives the music they hear. I created this website to catalog some of my personal favorite album covers.

Note: I know nothing about the art of coding. I used ChatGPT to generate the code for this website. I am hosting this website for free on GitHub Pages.
